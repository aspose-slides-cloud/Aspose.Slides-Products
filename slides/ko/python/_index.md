<p>Python용 Aspose.Slides Cloud SDK는 Aspose.Slides REST API를 기반으로 구축되었으며 MIT 라이선스에 따라 제공됩니다. Python용 Aspose.Slides Cloud SDK는 Python 애플리케이션에 쉽게 통합될 수 있으며 타사 설치 없이 클라우드에서 PowerPoint 및 OpenOffice 프레젠테이션 슬라이드를 가져오고 처리하고 변환할 수 있습니다.</p>
<p>Python용 Aspose.Slides Cloud SDK를 사용하면 Python 프로그램이 프레젠테이션에서 개별 슬라이드를 다운로드하고, 슬라이드에서 자리 표시자 정보를 가져오고, 슬라이드를 복사, 이동 또는 삭제할 수 있습니다. Python SDK를 사용하여 슬라이드의 글꼴 구성표, 형식 구성표, 색 구성표 또는 테마를 가져올 수도 있습니다. 클라우드에서 호스팅되는 프레젠테이션에서 마스터 슬라이드 정보를 복제하기 위한 지원도 제공됩니다.</p>
<p>Python용 Aspose.Slides Cloud SDK를 사용하면 Python 앱이 슬라이드 이미지, 도형, 메모, 텍스트 발생 및 기타 문서 속성과 함께 작동할 수 있습니다. Python SDK를 사용하여 PPT, PPTX, ODP, OTP를 PDF, XPS, SWF, HTML 또는 다양한 이미지 형식(BMP, JPEG, TIFF, SVG, PNG)으로 쉽게 변환할 수 있습니다.</p>
