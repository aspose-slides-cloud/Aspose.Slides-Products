<p>Go용 Aspose.Slides Cloud SDK는 클라우드에서 호스팅되는 Microsoft PowerPoint 및 OpenOffice 프레젠테이션 슬라이드를 처리하고 조작하는 Golang 프로그래머를 지원하기 위해 개발되었습니다. 이 클라우드 SDK는 Aspose.Slides REST API를 기반으로 하며 타사 소프트웨어를 설치하지 않고도 작동합니다. Go용 Aspose.Slides Cloud SDK는 MIT 라이선스에 따라 제공됩니다.</p>
<p>Go용 Aspose.Slides Cloud SDK를 사용하면 프레젠테이션 뷰어, 제작자 및 변환기 기능을 Go 프로그래밍 언어로 개발된 클라우드 기반 앱에 원활하게 통합할 수 있습니다. 이렇게 하면 Golang 코드가 새 프레젠테이션을 만들거나 클라우드 스토리지에서 기존 프레젠테이션 슬라이드에 액세스할 수 있습니다. Go 프로그램은 전체 프레젠테이션, 개별 슬라이드, 메모, 테마, 애니메이션, 도형, 이미지, 텍스트 및 사용자 정의 글꼴로 작업할 수 있습니다.</p>
<p>Go용 Aspose.Slides Cloud SDK를 사용하면 Go 애플리케이션에서 Microsoft PowerPoint(PPT, PPTX) 및 OpenOffice(ODP, OTP) 형식을 지원되는 다양한 형식(예: PDF, SWF, XPS, HTML, BMP, PNG, JPEG, TIFF 및 SVG.</p>
