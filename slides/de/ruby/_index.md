<p>Aspose.Slides Cloud SDK für Ruby ermöglicht Computerprogrammierern das Generieren, Modifizieren, Rendern und Konvertieren von PowerPoint- und OpenOffice-Präsentationen ohne Verwendung von Microsoft PowerPoint. Das SDK bietet Unterstützung für die meisten Microsoft PowerPoint-Dateiformate wie POT, PPT, PPS, POTX, PPTX, PPSX und ODP und exportiert diese in gängige Formate.</p>
<p>Aspose.Slides Cloud SDK für Ruby enthält Unterstützung für viele erweiterte und grundlegende Funktionen wie das Erstellen und Ändern von Präsentationen, das Zusammenführen und Aufteilen von Präsentationsdateien, das Exportieren von PowerPoint- und OpenOffice-Präsentationen, das Hinzufügen neuer Folien, das Generieren von Präsentationen aus einer Datenbank, das Hinzufügen von Layoutfolien zu Präsentationen, Fügen Sie Bilder zu Präsentationen hinzu und vieles mehr.</p>
<p>Sie können Ihre Präsentationen mit nur ein paar Befehlen in Ihren eigenen Ruby-Anwendungen in mehrere gängige Dateiformate konvertieren. Das SDK unterstützt die reibungslose Konvertierung in zahlreiche Dateiformate wie PPT, POT, PPS, PPTX, POTX, PPSX, PPTM, PPSM, POTM, ODP, OTP, TIFF, PDF, XPS, JPEG, PNG, GIF, BMP, SVG, SWF, HTML und viele mehr.</p>

    <h2 class="h2title">
     Konvertierung von PowerPoint-Dateien in PDF über Ruby
    </h2>
    <p>
     Aspose.Slides Cloud SDK für Ruby bietet Funktionen, mit denen Softwareentwickler PowerPoint-Präsentationen mit nur wenigen Befehlen in ihren eigenen Anwendungen programmgesteuert in PDF-Dateiformate konvertieren können. Das Portable Document Format (PDF) ist aus zahlreichen Gründen eines der am häufigsten verwendeten und beliebtesten Dateiformate, z. B. zur Verringerung der Dateigröße, vertrauenswürdige Sicherheit, universelle Verwendung und mehr. Das Tolle am PDF ist, dass es unabhängig von Hardware, Betriebssystem und Anwendungssoftware verwendet werden kann.
    </p>

Mehrere wichtige Funktionen wie das Einfügen neuer Folien, das Ändern und Kopieren von Folien, der Folienmoment an einer neuen Position, das Löschen von Folien, das Verwalten des Folienhintergrunds, das Hinzufügen von Folienkommentaren und vieles mehr. Das SDK bietet auch Funktionen zum Exportieren Ihrer Folien in Bilder und andere unterstützte Dateiformate.
    </p>

Mit ein paar Ruby-Befehlen können Sie ganz einfach Bilder in den Folien Ihrer Präsentation hinzufügen und ändern. Es unterstützt auch Funktionen wie das Extrahieren von Bildern aus einer PPT-Präsentation, das Abrufen von Bildinformationen, das Extrahieren von Bildern in einem bestimmten Format und vieles mehr.
