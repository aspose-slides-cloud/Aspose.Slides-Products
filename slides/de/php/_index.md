<p>Aspose.Slides Cloud SDK für PHP hilft Ihnen beim Erstellen von Cloud-basierten PHP-Anwendungen, die PPT-Folien und PowerPoint-Präsentationen erstellen können, ohne Software zu installieren. Aspose.Slides Cloud SDK für PHP unterstützt auch OpenOffice-Präsentationsformate, ODP & OTP. Aspose.Slides Cloud SDK für PHP basiert auf der Aspose.Slides REST API und wird unter einer MIT-Lizenz angeboten.</p>
<p>Mit Aspose.Slides Cloud SDK für PHP können Ihre Cloud-PHP-Programme neue Präsentationen erstellen oder auf vorhandene Präsentationen zugreifen, die sich im Cloud-Speicher befinden. Über unser PHP REST SDK können Sie mit der gesamten Präsentation arbeiten, einzelne Folien ändern, die Masterfolie klonen, die interaktive Sequenz der PowerPoint-Präsentation aktualisieren und benutzerdefinierte Schriftarten angeben, die in der Präsentation verwendet werden sollen.</p>
<p>Aspose.Slides Cloud SDK für PHP unterstützt Ihren PHP-Code beim Extrahieren von Formen, Bildern und Notizen aus Präsentationsfolien. Sie können PPTX, PPT, ODP und OTP auch in die Formate SWF, HTML, BMP, PNG, TIFF, JPEG und SVG konvertieren.</p>
