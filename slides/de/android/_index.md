{{< blocks/products/pf/i18n/tab-content >}}
<p>Aspose.Slides Cloud Android SDK gibt Softwareentwicklern die Möglichkeit, Funktionen zum Erstellen, Ändern und Konvertieren von Microsoft PowerPoint- und OpenOffice-Präsentationen in ihre eigenen Android-Anwendungen zu integrieren. Aspose.Slides Cloud Android SDK ist ein Wrapper um die Aspose.Slides REST API und wird unter einer MIT-Lizenz angeboten. Es bietet Unterstützung für mehrere wichtige Dateiformate wie POT, PPT, PPS, POTX, PPTX, PPSX und ODP und exportiert diese in gängige Formate.</p>
<p>Das Cloud SDK bietet volle Unterstützung für zahlreiche präsentationsbezogene Funktionen wie das Erstellen und Bearbeiten von Präsentationen, das Exportieren von Präsentationen in andere unterstützte Dateiformate, das Zusammenführen und Teilen von Präsentationen, das Hinzufügen und Verwalten von Folien, das Hinzufügen von Folienlayouts zu Präsentationen, das Einfügen von Bildern in Präsentationen und vieles mehr. Sie können Ihre Präsentationen in verschiedene gängige Dateiformate wie PPT, POT, PPS, PPTX, POTX, PPSX, PPTM, PPSM, POTM, ODP, OTP, TIFF, PDF, XPS, JPEG, PNG, GIF, BMP, SVG, SWF, HTML und viele mehr.</p>

    </h2>

Das SDK unterstützt die Konvertierung in Formate wie TIFF, PDF, PPTX, XPS und PPT usw. Sie können Inhalte wie Text und Bilder aus den PowerPoint-Dokumenten extrahieren.
    </p>
   </div>
   <div class="col-lg-12">
    <h2 class="h2title">
     Arbeiten mit PowerPoint-Präsentationsfolien in Android
    </h2>
    <p>
     Aspose.Slides Cloud Android SDK bietet Computerprogrammierern die Möglichkeit, mit Präsentationsfolien in ihren eigenen Android-Anwendungen zu arbeiten. Das SDK bietet auch volle Unterstützung für mehrere wichtige Funktionen, wie das Hinzufügen neuer Folien, das Kopieren von Folien, das Verwalten von Folienkommentaren, das Verschieben von Folien an eine neue Position, das Löschen von Folien, die Verwaltung des Folienhintergrunds und vieles mehr. Darüber hinaus können Sie Ihre Folien auch in Bilddateiformate sowie mehrere andere Dateiformate exportieren.
    </p>
    <h2 class="h2title">
     Arbeiten mit Formen in Präsentationen
    </h2>
    <p>
     Aspose.Slides Cloud Android SDK ermöglicht es Softwareentwicklern, mit Formen in ihren eigenen Anwendungen zu arbeiten. Das Android SDK unterstützt mehrere wichtige Funktionen im Zusammenhang mit der Verwaltung von Formen, z. B. das Extrahieren von Formen aus einer Folie, das Abrufen einer bestimmten Form von einer Folie, das Hinzufügen eines Formobjekts zu einer Gruppenform, das Löschen eines Formobjekts aus einer Gruppenform, das Arbeiten mit Formen mithilfe von Speicher von Drittanbietern, Arbeiten mit Smart Art Shape Management und vieles mehr.
    </p>
    <h2 class="h2title">

    </h2>
    <p>

Entwickler können sowohl das FontScheme als auch das FormatScheme einer Folie in einer PowerPoint-Präsentation abrufen. Sie können ColorScheme-Informationen lesen oder ganz einfach das Design einer Folie in einer PowerPoint-Präsentation abrufen.
