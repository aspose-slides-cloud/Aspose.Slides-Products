<p>Aspose.Slides Cloud SDK für .NET wird verwendet, um PPT-, PPTX-, ODP- und OTP-Verarbeitungsfunktionen nahtlos in Ihre Cloud-basierten Anwendungen zu integrieren, die mit C#, ASP.NET oder anderen .NET-unterstützten Sprachen erstellt wurden. Aspose.Slides Cloud SDK für .NET ist ein Wrapper um die Aspose.Slides REST API und wird unter einer MIT-Lizenz angeboten.</p>
<p>Aspose.Slides Cloud SDK für .NET verbessert Ihren C#-Code, um PowerPoint-Präsentationen zu erstellen, ausgewählte Folien zusammenzuführen, Präsentationen aufzuteilen sowie PowerPoint- und OpenOffice-Präsentationsdokumente in PDF, XPS, JPEG, PNG, BMP, TIFF, SVG, HTML und SWF.</p>
<p>Aspose.Slides Cloud SDK für .NET ermöglicht es Ihrer Cloud-basierten .NET-Anwendung auch, mit Präsentationsfolienanimationen zu arbeiten, Masterfolieninformationen zu lesen und zu klonen, Foliennotizen zu extrahieren und herunterzuladen, Präsentationsbilder in ein bestimmtes Format zu extrahieren, Formen aus einer Folie zu extrahieren , sowie verschiedene andere Operationen mit Ihren C#-Programmen über unser Aspose.Slides Cloud SDK für .NET durchführen.</p>
