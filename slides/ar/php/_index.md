<p>يساعدك Aspose.Slides Cloud SDK for PHP على إنشاء تطبيقات PHP مستندة إلى السحابة يمكنها إنشاء شرائح PPT وعروض PowerPoint التقديمية دون تثبيت أي برنامج. يدعم Aspose.Slides Cloud SDK لـ PHP أيضًا تنسيقات العرض التقديمي OpenOffice وODP وOTP. يعتمد Aspose.Slides Cloud SDK لـ PHP على Aspose.Slides REST API ويتم تقديمه بموجب ترخيص MIT.</p>
<p>باستخدام Aspose.Slides Cloud SDK لـ PHP، ستتمكن برامج PHP السحابية من إنشاء عروض تقديمية جديدة أو الوصول إلى العروض التقديمية الموجودة في وحدة التخزين السحابية. من خلال PHP REST SDK، يمكنك العمل مع العرض التقديمي بأكمله، وتعديل الشرائح الفردية، واستنساخ الشريحة الرئيسية، وتحديث التسلسل التفاعلي لعرض PowerPoint التقديمي، وتحديد الخطوط المخصصة لاستخدامها في العرض التقديمي.</p>
<p>يساعد Aspose.Slides Cloud SDK for PHP كود PHP الخاص بك على استخراج الأشكال والصور والملاحظات من شرائح العرض التقديمي. يمكنك أيضًا تحويل PPTX وPPT وODP وOTP إلى تنسيقات SWF وHTML وBMP وPNG وTIFF وJPEG وSVG.</p>
