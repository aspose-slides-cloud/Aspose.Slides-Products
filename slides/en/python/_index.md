<p>Aspose.Slides Cloud SDK for Python is built on top of Aspose.Slides REST API and is offered under an MIT license. Aspose.Slides Cloud SDK for Python can be easily integrated into your Python applications and enable them to fetch, process and convert PowerPoint as well as OpenOffice presentation slides in the cloud, without any third-party installation.</p>
<p>Using Aspose.Slides Cloud SDK for Python, your Python programs will be able to download individual slide from a presentation, get placeholder information from a slide, copy, move or delete a slide. You can also get font scheme, format scheme, color scheme or theme of a slide using Python SDK. Support is also available for cloning master slide information from a presentation hosted on the cloud.</p>
<p>Aspose.Slides Cloud SDK for Python also allows your Python apps to work with slide images, shapes, notes, text occurrences and other document attributes. You can easily utilize Python SDK to convert PPT, PPTX, ODP, OTP to PDF, XPS, SWF, HTML or many of the image formats (BMP, JPEG, TIFF, SVG, PNG).</p>
