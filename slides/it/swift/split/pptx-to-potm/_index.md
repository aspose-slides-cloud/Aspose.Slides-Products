1. Crea un account su <a href="https://dashboard.aspose.cloud/">Dashboard</a> per ottenere quote API gratuite e dettagli di autorizzazione
1. Inizializza ```SlidesApi``` con ID client e Segreto client
1. Chiama uno dei metodi di suddivisione, a seconda di dove si trova il file PPTX di input e dove desideri archiviare i file POTM risultanti
    - ```split``` per dividere un file nella memoria e salvare il risultato nella memoria. Ciascuna diapositiva del PPTX di input viene salvata come POTM separato nella cartella di archiviazione specificata (o nella cartella del file di input per impostazione predefinita).
    - ```splitOnline``` per dividere un file locale e scaricare il risultato come archivio ZIP. Ogni diapositiva dell'input PPTX viene scaricata come POTM separata all'interno dell'archivio ZIP.
    - ```splitAndSaveOnline``` per dividere un file locale e salvare il risultato nella memoria. Ogni diapositiva dell'input PPTX viene salvata come POTM separata nella cartella di archiviazione specificata (o nella cartella principale per impostazione predefinita).
