1. Inizializza ```SlidesApi``` con ID client e Segreto client
1. Chiama uno dei metodi di suddivisione, a seconda di dove si trova il file POTM di input e dove desideri archiviare i file JPEG risultanti
    - ```Split``` per dividere un file nella memoria e salvare il risultato nella memoria. Ciascuna diapositiva del POTM di input viene salvata come JPEG separato nella cartella di archiviazione specificata (o nella cartella del file di input per impostazione predefinita).
    - ```SplitOnline``` per dividere un file locale e scaricare il risultato come archivio ZIP. Ogni diapositiva dell'input POTM viene scaricata come JPEG separata all'interno dell'archivio ZIP.
    - ```SplitAndSaveOnline``` per dividere un file locale e salvare il risultato nella memoria. Ogni diapositiva dell'input POTM viene salvata come JPEG separata nella cartella di archiviazione specificata (o nella cartella principale per impostazione predefinita).
