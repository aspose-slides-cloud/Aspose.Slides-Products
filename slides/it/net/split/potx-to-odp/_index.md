1. Inizializza ```SlidesApi``` con ID client e Segreto client
1. Chiama uno dei metodi di suddivisione, a seconda di dove si trova il file POTX di input e dove desideri archiviare i file ODP risultanti
    - ```Split``` per dividere un file nella memoria e salvare il risultato nella memoria. Ciascuna diapositiva del POTX di input viene salvata come ODP separato nella cartella di archiviazione specificata (o nella cartella del file di input per impostazione predefinita).
    - ```SplitOnline``` per dividere un file locale e scaricare il risultato come archivio ZIP. Ogni diapositiva dell'input POTX viene scaricata come ODP separata all'interno dell'archivio ZIP.
    - ```SplitAndSaveOnline``` per dividere un file locale e salvare il risultato nella memoria. Ogni diapositiva dell'input POTX viene salvata come ODP separata nella cartella di archiviazione specificata (o nella cartella principale per impostazione predefinita).
