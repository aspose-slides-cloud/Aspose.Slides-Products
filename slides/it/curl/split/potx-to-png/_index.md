1. Chiama uno dei metodi di suddivisione, a seconda di dove si trova il file POTX di input e dove desideri archiviare i file PNG risultanti
    - ```Split``` per dividere un file nella memoria e salvare il risultato nella memoria. Ciascuna diapositiva del POTX di input viene salvata come PNG separato nella cartella di archiviazione specificata (o nella cartella del file di input per impostazione predefinita).

    - ```SplitAndSaveOnline``` per dividere un file locale e salvare il risultato nella memoria. Ogni diapositiva dell'input POTX viene salvata come PNG separata nella cartella di archiviazione specificata (o nella cartella principale per impostazione predefinita).
