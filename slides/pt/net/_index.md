<p>O Aspose.Slides Cloud SDK para .NET é usado para integrar perfeitamente os recursos de processamento de PPT, PPTX, ODP e OTP em seus aplicativos baseados em nuvem criados usando C#, ASP.NET ou outras linguagens compatíveis com .NET. Aspose.Slides Cloud SDK para .NET é um wrapper em torno da API REST Aspose.Slides e é oferecido sob uma licença MIT.</p>
<p>Aspose.Slides Cloud SDK para .NET aprimora seu código C# para criar apresentações em PowerPoint, mesclar slides selecionados, dividir apresentações, bem como converter documentos de apresentação em PowerPoint e OpenOffice para PDF, XPS, JPEG, PNG, BMP, TIFF, SVG, HTML e SWF.</p>
<p>Aspose.Slides Cloud SDK para .NET também permite que seu aplicativo .NET baseado em nuvem trabalhe com animações de slides de apresentação, leia e clone informações de slides mestres, extraia e baixe notas de slides, extraia imagens de apresentação para um formato específico, extraia formas de um slide , além de executar várias outras operações usando seus programas C# por meio de nosso Aspose.Slides Cloud SDK para .NET.</p>
