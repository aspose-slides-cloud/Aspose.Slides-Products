<p>Aspose.Slides Cloud SDK para PHP ajuda você a criar aplicativos PHP baseados em nuvem que podem criar slides PPT e apresentações em PowerPoint sem instalar nenhum software. Aspose.Slides Cloud SDK para PHP também suporta formatos de apresentação OpenOffice, ODP e OTP. Aspose.Slides Cloud SDK para PHP é baseado na API REST Aspose.Slides e é oferecido sob uma licença MIT.</p>
<p>Usando Aspose.Slides Cloud SDK para PHP, seus programas PHP em nuvem seriam capazes de criar novas apresentações ou acessar apresentações existentes localizadas no armazenamento em nuvem. Por meio de nosso PHP REST SDK, você pode trabalhar com toda a apresentação, modificar slides individuais, clonar slide mestre, atualizar a sequência interativa da apresentação do PowerPoint e especificar fontes personalizadas a serem usadas na apresentação.</p>
<p>Aspose.Slides Cloud SDK para PHP ajuda seu código PHP a extrair formas, imagens e notas dos slides da apresentação. Você também pode converter PPTX, PPT, ODP e OTP para os formatos SWF, HTML, BMP, PNG, TIFF, JPEG e SVG.</p>
