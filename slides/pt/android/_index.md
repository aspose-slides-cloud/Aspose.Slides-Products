<p>Aspose.Slides Cloud Android SDK oferece aos desenvolvedores de software o poder de incorporar recursos de criação, modificação e conversão de apresentações do Microsoft PowerPoint e OpenOffice em seus próprios aplicativos Android. Aspose.Slides Cloud Android SDK é um wrapper em torno da API REST Aspose.Slides e é oferecido sob uma licença MIT. Ele fornece suporte para vários formatos de arquivo importantes, como POT, PPT, PPS, POTX, PPTX, PPSX e ODP e os exporta para formatos populares.</p>
<p>O Cloud SDK fornece suporte completo para vários recursos relacionados a apresentações, como criar e editar apresentações, exportar apresentações para outros formatos de arquivo compatíveis,  mesclar e dividir apresentações,  adição e gerenciamento de slides, adição de layout de slides a apresentações, inserção de imagens em apresentações e muito mais. Você pode converter suas apresentações em vários formatos de arquivo populares, como PPT, POT, PPS, PPTX, POTX, PPSX, PPTM, PPSM, POTM, ODP, OTP, TIFF, PDF, XPS, JPEG, PNG, GIF, BMP, SVG, SWF, HTML e muito mais.</p>
