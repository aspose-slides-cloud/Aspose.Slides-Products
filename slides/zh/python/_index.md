<p>适用于 Python 的 Aspose.Slides Cloud SDK 建立在 Aspose.Slides REST API 之上，并在 MIT 许可下提供。 Aspose.Slides Cloud SDK for Python 可以轻松集成到您的 Python 应用程序中，使它们能够在云端获取、处理和转换 PowerPoint 以及 OpenOffice 演示幻灯片，无需任何第三方安装。</p>
<p>适用于 Python 的 Aspose.Slides Cloud SDK，您的 Python 程序将能够从演示文稿中下载单个幻灯片、从幻灯片中获取占位符信息、复制、移动或删除幻灯片。 您还可以使用 Python SDK 获取幻灯片的字体方案、格式方案、配色方案或主题。 还支持从云托管的演示文稿中克隆母版幻灯片信息。</p>
<p>适用于 Python 的 Aspose.Slides Cloud SDK 还允许您的 Python 应用程序处理幻灯片图像、形状、注释、文本出现和其他文档属性。 您可以轻松地利用 Python SDK 将 PPT、PPTX、ODP、OTP 转换为 PDF、XPS、SWF、HTML 或许多图像格式（BMP、JPEG、TIFF、SVG、PNG）。</p>
