<p>Aspose.Slides Cloud SDK pour .NET est utilisé pour intégrer de manière transparente les capacités de traitement PPT, PPTX, ODP, OTP dans vos applications basées sur le cloud créées à l'aide de C #, ASP.NET ou d'autres langages pris en charge par .NET. Aspose.Slides Cloud SDK pour .NET est un wrapper autour de l'API REST Aspose.Slides et est proposé sous une licence MIT.</p>
<p>Aspose.Slides Cloud SDK pour .NET améliore votre code C # pour créer des présentations PowerPoint, fusionner des diapositives sélectionnées, diviser des présentations, ainsi que convertir des documents de présentation PowerPoint et OpenOffice en PDF, XPS, JPEG, PNG, BMP, TIFF, SVG, HTML et SWF.</p>
<p>Aspose.Slides Cloud SDK pour .NET permet également à votre application .NET basée sur le cloud de travailler avec des animations de diapositives de présentation, de lire et de cloner des informations sur la diapositive principale, d'extraire et de télécharger des notes de diapositive, d'extraire l'image de présentation dans un format particulier, d'extraire des formes d'une diapositive , ainsi que d'effectuer diverses autres opérations à l'aide de vos programmes C# via notre Aspose.Slides Cloud SDK pour .NET.</p>
