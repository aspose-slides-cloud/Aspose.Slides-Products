<p>Aspose.Slides Cloud SDK para PHP lo ayuda a crear aplicaciones PHP basadas en la nube que pueden crear diapositivas PPT y presentaciones de PowerPoint sin instalar ningún software. Aspose.Slides Cloud SDK para PHP también es compatible con los formatos de presentación de OpenOffice, ODP y OTP. Aspose.Slides Cloud SDK para PHP se basa en la API REST de Aspose.Slides y se ofrece bajo una licencia MIT.</p>
<p>Usando Aspose.Slides Cloud SDK para PHP, sus programas PHP en la nube podrían crear nuevas presentaciones o acceder a presentaciones existentes ubicadas en el almacenamiento en la nube. A través de nuestro PHP REST SDK, puede trabajar con toda la presentación, modificar diapositivas individuales, clonar diapositivas maestras, actualizar la secuencia interactiva de la presentación de PowerPoint y especificar fuentes personalizadas para usar en la presentación.</p>
<p>Aspose.Slides Cloud SDK para PHP ayuda a su código PHP a extraer formas, imágenes y notas de las diapositivas de la presentación. También puede convertir PPTX, PPT, ODP y OTP a formatos SWF, HTML, BMP, PNG, TIFF, JPEG, SVG.</p>
