{{% blocks/products/pf/agp/feature-section-col title="Dividir OTP en PDF con Perl" %}}
1. Cree una cuenta en <a href="https://dashboard.aspose.cloud/">el panel de control</a> para obtener una cuota de API gratuita y detalles de autorización
1. Inicialice ```SlidesApi``` con ID de cliente y Secreto de cliente
1. Llame a uno de los métodos de división, según dónde esté su archivo OTP de entrada y dónde desee almacenar los archivos PDF resultantes
    - ```split``` para dividir un archivo en el almacenamiento y guardar el resultado en el almacenamiento. Cada diapositiva del OTP de entrada se guarda como un PDF independiente en la carpeta de almacenamiento especificada (o en la carpeta de archivos de entrada de forma predeterminada).
    - ```split_online``` para dividir un archivo local y descargar el resultado como un archivo ZIP. Cada diapositiva del OTP de entrada se descargará como un PDF separado dentro del archivo ZIP.
    - ```split_and_save_online``` para dividir un archivo local y guardar el resultado en el almacenamiento. Cada diapositiva del OTP de entrada se guardará como un PDF independiente en la carpeta de almacenamiento especificada (o en la carpeta raíz de forma predeterminada).
