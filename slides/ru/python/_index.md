<p>Aspose.Slides Cloud SDK для Python построен на основе Aspose.Slides REST API и предлагается по лицензии MIT. Aspose.Slides Cloud SDK для Python можно легко интегрировать в ваши приложения Python и позволить им извлекать, обрабатывать и преобразовывать PowerPoint, а также слайды презентаций OpenOffice в облаке без какой-либо сторонней установки.</p>
<p>Используя Aspose.Slides Cloud SDK для Python, ваши программы Python смогут загружать отдельные слайды из презентации, получать информацию о заполнителях из слайда, копировать, перемещать или удалять слайд. Вы также можете получить схему шрифта, схему формата, цветовую схему или тему слайда с помощью Python SDK. Также доступна поддержка клонирования информации мастер-слайда из презентации, размещенной в облаке.</p>
<p>Aspose.Slides Cloud SDK для Python также позволяет вашим приложениям Python работать с изображениями слайдов, формами, заметками, вхождениями текста и другими атрибутами документа. Вы можете легко использовать Python SDK для преобразования PPT, PPTX, ODP, OTP в PDF, XPS, SWF, HTML или многие форматы изображений (BMP, JPEG, TIFF, SVG, PNG).</p>
