<p>Aspose.Slides Cloud Android SDK дает разработчикам программного обеспечения возможность включать возможности создания, изменения и преобразования презентаций Microsoft PowerPoint и OpenOffice в свои собственные приложения для Android. Aspose.Slides Cloud Android SDK представляет собой оболочку Aspose.Slides REST API и предлагается по лицензии MIT. Он обеспечивает поддержку нескольких важных форматов файлов, таких как POT, PPT, PPS, POTX, PPTX, PPSX и ODP, и экспортирует их в популярные форматы.</p>
<p>Cloud SDK обеспечивает полную поддержку многочисленных функций, связанных с презентациями, таких как создание и редактирование презентаций, экспорт презентаций в другие поддерживаемые форматы файлов, объединение и разделение презентаций, добавление слайдов и управление ими, добавление макета слайдов в презентации, вставка изображений в презентации и многое другое. Вы можете конвертировать свои презентации в несколько популярных форматов файлов, таких как PPT, POT, PPS, PPTX, POTX, PPSX, PPTM, PPSM, POTM, ODP, OTP, TIFF, PDF, XPS, JPEG, PNG, GIF, BMP, SVG, SWF, HTML и многое другое.</p>
